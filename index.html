الخطوة الأولي رفع الملفات

اي موقع علي حسب ما اعلم سوف يكون بداخل مجلد public_html لذلك عليك فقط رفع الملف المضغوط علي السيرفر

بعد أكتمال رفع الملف بداخل المجلد قم بفك الضغط عن المجلد

سوف يكون شكل الملفات هكذا

بعد ذلك عليك نقل الملفات الي المجلد الرئيسيى هكذا

بعد ذلك قم بحذف الملف المضغوط

الخطوة الثانية قاعدة البيانات

اذهب الي لوحة التحكم الخاصه بالاستضافه واختر قواعد البيانات ثم قم بانشاء قاعدة بيانات جديدة

أرجع مرة أخري الي ملفات الموقع ثم قم بفتح ملف يحمل الامتداد .env

قم بكتابة بيانات الاتصال بقاعدة البيانات [ اسم المستخدمDB_USERNAME - كلمة المرور DB_PASSWORD - أسم قاعدة البيانات DB_DATABASE ] وقم بحفظها بالنقر علي زر حفظ

يجب أن تغير صلاحيات المستخدم الي اعطاءها كل الصلاحيات

بعد ذلك توجة علي الي لوحة تحكم الأستضافة ثم قم بالدخول الي php my admin

قم بأختيار قاعدة البيانات ثم أضغط أستيراد

قم بأختيار ملف قاعدة البيانات وأضغط أستيراد

بعد انتهاء الاستيراد سوف تظهر لك رساله بنجاح العملية وبنية قاعدة البيانات
